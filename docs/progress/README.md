# 研究进展

> 本目录用于记录博士课题各阶段进展，通过 Git 在 GitHub 与两台本地电脑之间同步。

## 文件说明

| 文件 | 用途 |
|------|------|
| [README.md](README.md) | 本文件：进展目录说明 |
| [milestones.md](milestones.md) | 对照 research_plan 各阶段的 checklist |
| [CHANGELOG.md](CHANGELOG.md) | 按日期记录每次 push 的进展摘要 |
| [logs/](logs/) | 可选：详细研究日志（按日/周） |

## 更新习惯

1. **开工前**：`git pull` 拉取最新进展  
2. **收工后**：更新 `CHANGELOG.md`（及必要时 `milestones.md`），然后 commit + push  
3. **格式示例**（CHANGELOG）：
   ```
   ## 2026-06-25
   - 完成综述 L1 初稿
   - 补充 FSA、PLENA 对比段落
   ```
